    What will be the value of x and y after executing the following java code snippet?
public class Test {
    public static void main(String[] args) {
        int x = 5;
        int y = x++ + ++x;
        System.out.println("x: " + x + ", y: " + y);
    }
}
    
a) x: 7, y: 11
b) x: 7, y: 12(ans)
c) x: 6, y: 12
d) Compilation Error
